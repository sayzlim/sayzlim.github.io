Repair Disk Permissions Myth

Thomas Brand addressed one of the most common myths on disk permissions: The only time you need to run Repair Disk Permissions is if you changed a system-level

Sayz Lim Aug 22, 2013
  • Link

Thomas Brand addressed one of the most common myths on disk permissions:

The only time you need to run Repair Disk Permissions is if you changed a system-level permission you shouldn’t have.

I used to stay up late searching for tools and tips to optimize Mac. I downloaded all the recommended applications that claim to optimize and speed up OS X. I guess what they do are merely clearing caches and changing the priority of the processes. It wasn’t a pleasant experience. I ended up staying awake the whole night trying to revert its condition from fast to normal.

Nowadays, I let OS X manages everything. I’ve only adjusted some animations to improve my experience.

Avoid needless hacks when OS X doesn’t freeze your Mac. Instead of spending your time optimizing OS X, try to come up with a way to finish your tasks efficiently.
